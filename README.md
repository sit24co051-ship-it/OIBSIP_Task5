# Stopwatch App

## Overview
Stopwatch App is an Android application developed using Java and XML in Android Studio. The application allows users to measure elapsed time accurately while providing additional customization and usability features.

## Features

- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Lap Time Recording
- Dark/Light Mode Toggle
- Custom Background Colors
  - Blue
  - Purple
  - Black
  - White
- Settings Persistence using SharedPreferences
- Simple and User-Friendly Interface

## Technologies Used

- Java
- XML
- Android Studio
- SharedPreferences
